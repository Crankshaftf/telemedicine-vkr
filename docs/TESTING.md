# Чеклист тестирования (контрольный сценарий)

Перед защитой ВКР пройдите сценарий **от начала до конца** с аккаунтом обычного пользователя `user@test.ru`.

## Подготовка

**Локальная сеть (Wi-Fi):**
- [ ] `.\scripts\start-network.ps1`
- [ ] `.\scripts\build-apk.ps1`

**Доступ из интернета (мобильный интернет на телефоне):**
- [ ] `.\scripts\start-public.ps1` — получите HTTPS-адрес
- [ ] `.\scripts\build-apk.ps1 -UsePublicUrl` — пересобрать APK
- [ ] На телефоне открыть **`/download/`** и установить без USB
- [ ] Подробнее: [DEPLOY-PUBLIC.md](DEPLOY-PUBLIC.md)

**Вручную (разработка):**
- [ ] Docker: `docker compose up -d`
- [ ] Backend: `uvicorn app.main:app --reload --host 0.0.0.0 --port 8000`
- [ ] Seed: `python scripts\seed.py`
- [ ] Android-эмулятор или телефон, backend доступен (`10.0.2.2:8000` или IP ПК)
- [ ] Веб-панель: http://localhost:8000/panel/

## Негативные проверки

- [ ] Неверный пароль → сообщение об ошибке
- [ ] Запись на занятый слот → ошибка 409 (через Swagger или повторный запрос)
- [ ] Пациент не видит чужие обращения
- [ ] Файл > 10 МБ → отклонён

---

## Критерии приёмки (из ТЗ)

- [ ] Регистрация и вход работают
- [ ] Выбор врача и времени работает
- [ ] Заявка сохраняется в БД
- [ ] Врач видит заявку и сохраняет заключение
- [ ] Пациент видит заключение
- [ ] ≥ 10 скриншотов готовы
- [ ] Приложение запускается на эмуляторе/телефоне
