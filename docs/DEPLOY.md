# Развёртывание в локальной сети

## Быстрый старт

### 1. Запустить сервер (на ПК)

```powershell
cd c:\Users\Crankshaft\Documents\telemedicine-vkr
.\scripts\start-network.ps1
```

Скрипт:
- определит IP в Wi-Fi (сейчас: **192.168.31.95**)
- откроет порт 8000 в firewall
- запустит API на `0.0.0.0:8000` (доступен из сети)
- если Docker недоступен — использует SQLite

**Ссылки с телефона/другого ПК:**
- API: http://192.168.31.95:8000/
- Swagger: http://192.168.31.95:8000/docs
- Панель врача: http://192.168.31.95:8000/panel/

> Замените IP на свой, если отличается. Телефон и ПК — **одна Wi-Fi сеть**.

### 2. APK уже собран

```
c:\Users\Crankshaft\Documents\telemedicine-vkr\MedConnect.apk
```

Размер ~57 МБ. API внутри APK: `http://192.168.31.95:8000/`

Пересобрать после смены IP:

```powershell
.\scripts\build-apk.ps1
```

### 3. Установить на телефон

**Способ A — вручную:**
1. Скопируйте `MedConnect.apk` на телефон (USB, Telegram, Google Drive)
2. Откройте файл → «Установить»
3. Разрешите установку из неизвестных источников

**Способ B — через USB (adb):**
```powershell
.\scripts\install-apk.ps1
```

## Docker (рекомендуется для постоянной работы)

1. Запустите **Docker Desktop**
2. `docker compose up -d --build`
3. API доступен на порту 8000

---

## Если телефон не подключается

1. Проверьте, что сервер запущен (в консоли «Uvicorn running»)
2. На телефоне откройте в браузере: http://192.168.31.95:8000/health
3. Запустите PowerShell **от администратора** и снова `start-network.ps1` (firewall)
4. Отключите изоляцию клиентов AP на роутере (если есть)
5. Пересоберите APK если IP ПК изменился
