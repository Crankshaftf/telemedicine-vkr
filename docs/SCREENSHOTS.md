# Список скриншотов для диплома (15 шт.)

Рекомендуется снимать на **эмуляторе Android** с разрешением 1080×2400 (типичный смартфон).

## Android-приложение (пациент) — минимум 10

| № | Экран | Аккаунт | Что показать |
|---|-------|---------|--------------|
| 1 | Splash | любой | Логотип «МедКоннект» |
| 2 | Вход | `user@test.ru` | Форма авторизации |
| 3 | Регистрация | новый / пустая форма | Поля ФИО, телефон, согласие |
| 4 | Главное меню | `user@test.ru` | 4 пункта меню, приветствие |
| 5 | Выбор специализации | `user@test.ru` | Список специальностей |
| 6 | Список врачей | `user@test.ru` | Карточка Петрова П.П. |
| 7 | Выбор времени | `user@test.ru` | Свободные слоты |
| 8 | Создание заявки | `user@test.ru` | Жалобы + прикреплённый файл |
| 9 | Подтверждение записи | `user@test.ru` | «Запись подтверждена» |
| 10 | История обращений | `user@test.ru` | Список записей |

## Советы для диплома

1. **Контрольный сценарий** — скриншоты 4–10 с `user@test.ru` (живой сценарий).
2. **Готовый результат** — скриншот 11 с `patient@test.ru` (заключение уже есть).
3. Сохраняйте в `docs/screenshots/` с именами: `01_splash.png`, `02_login.png`, …
4. Подписи к рисункам в дипломе: «Рисунок X — Экран … приложения МедКоннект».

## Папка для скриншотов

```
docs/screenshots/
├── 01_splash.png
├── 02_login.png
└── ...
```

Создайте папку вручную при съёмке или скопируйте из эмулятора:  
**Device File Explorer** → `/sdcard/Pictures/` или через кнопку камеры эмулятора.
