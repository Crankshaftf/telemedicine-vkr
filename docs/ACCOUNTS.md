## Создание / обновление аккаунтов

```powershell
cd backend
python scripts\seed.py
```

Скрипт идемпотентный: можно запускать повторно — добавит недостающих пользователей, не дублируя данные.

## Регистрация нового пользователя

Через Android: экран «Регистрация» или `POST /auth/register` в Swagger.

Пример тела запроса:

```json
{
  "full_name": "Новый Пользователь",
  "phone": "+79991234567",
  "email": "newuser@test.ru",
  "password": "password123",
  "consent": true
}
```
